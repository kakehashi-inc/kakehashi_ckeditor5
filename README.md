# CKEditor 5の専用カスタマイズ

CKEditor 5に個別の追加機能や、機能の置き換えを行うためのプロジェクトです。

## ビルド

```bash
yarn build
# npm build
```

./buildに出力される

## パッケージング

```bash
npm pack
# yarn pack
```

## 利用方法

パッケージングしたtgzをローカルノードとして他のプロジェクトで利用。
